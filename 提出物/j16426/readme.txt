◇ 概要 ◇
人vs人、人vsCPUの対戦ができるビリヤードゲームです。
球が3D表示されるなど、リアリティにこだわりました。


◇ ファイル構成 ◇
billiard.c … メインのソースファイル
billiard.h … メインのヘッダファイル
table.c    … 台のソースファイル
table.h    … 台のヘッダファイル
ball.c     … 球のソースファイル
ball.h     … 球のヘッダファイル
vector.c   … ベクトルのソースファイル

shape.c    … 図形描画のソースファイル
shape.h    … 図形描画のヘッダファイル
icon.ico   … アイコンの画像
icon.rc    … アイコン用のリソース
Makefile   … make用のルールを記述したファイル
readme.txt … 本ファイル


◇ 動作確認環境 ◇
・Windows 10 Home 64bit
・Cygwin 3.0.7
・GCC 7.4.0
・GLUT 3.7


◇ ビルドと実行方法 ◇
GLUTとglpngが導入されているCygwin上で、次のコマンドを入力することでビルド、実行することができます。

$ make
$ ./j16426.exe


◇ 注意事項 ◇
処理の都合上、ウィンドウサイズが大きいほど綺麗に描画されます。
ウィンドウを最大化してプレイすることをおすすめします。


◇ 遊び方 ◇
手球(白い玉)をキューで弾きます。
マウスを押している間キューが引かれ、離すとショットを打ちます。

ルールは一般的なナインボールと同様です。
残っている球のうち、一番数字が小さい球に最初に手球を当てなければいけません (次に当てるべき球は光ります)。
無効な球に当ててしまったり、手球をポケットに落としてしまうとファウルになります。
ファウルになった場合、次のプレイヤーは手球を好きな位置に配置することができます。

順番に球をポケットに落としていき、最終的に9番球を落としたプレイヤーの勝利です。
途中で9番球が落ちた場合、その時点で落としたプレイヤーの勝利になります。
